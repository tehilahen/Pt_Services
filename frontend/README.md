# 🛡️ PT Service Frontend - מערכת Security Scans

ממשק משתמש מודרני לניהול ומעקב אחר סריקות אבטחה וממצאים במערכות הארגון.

## 🚀 הפעלה מהירה

### דרישות מערכת
- Node.js 14+
- npm או yarn
- חיבור לשרת Backend (Python Flask)

### התקנה והפעלה

1. **התקנת תלויות:**
   ```bash
   npm install
   ```

2. **הפעלת הפרונטאנד:**
   ```bash
   npm start
   ```

3. **גישה למערכת:**
   - פתח דפדפן: `http://localhost:3000`
   - **שם משתמש:** `tehilaco`
   - **סיסמה:** הסיסמה שהגדרת (או `1234` אם לא שינית)

> ⚠️ **חשוב:** וודא שהשרת Backend רץ על פורט 5000!

## 🔌 חיבור לבקאנד

הפרונטאנד מתחבר לבקאנד על: `http://localhost:5000`

### API Endpoints:

**מערכות וסריקות:**
- `GET /api/systems` - רשימת כל המערכות
- `GET /api/systems/:id` - פרטי מערכת ספציפית
- `GET /api/scans` - רשימת כל הסריקות
- `GET /api/vulnerabilities` - כל הממצאים
- `GET /api/stats` - סטטיסטיקות כלליות

**אימות ואבטחה:**
- `POST /api/auth/login` - התחברות משתמש
- `POST /api/auth/logout` - התנתקות
- `POST /api/auth/forgot-password` - בקשת איפוס סיסמה (שולח מייל)
- `POST /api/auth/reset-password` - איפוס סיסמה עם טוקן

**בקשות:**
- `POST /api/scan-request` - שליחת בקשה לסריקת מערכת (שולח מייל מעוצב)

## 🐛 פתרון בעיות

### שגיאות נפוצות

1. **"Module not found"**
   ```bash
   npm install
   ```

2. **"Port 3000 is already in use"**
   ```bash
   PORT=3001 npm start
   ```

3. **"Backend connection failed" / שגיאת 401**
   - ודא שהשרת Backend רץ: `cd ../PT_Service_Backend && python main.py`
   - בדוק שהפורט 5000 פתוח
   - נקה cache: Ctrl+Shift+R בדפדפן

4. **"לא מצליח להתחבר"**
   - נסה איפוס סיסמה דרך "שכחתי סיסמא"
   - בדוק שהאימייל שלך במסד הנתונים
   - ודא שהסיסמה עומדת בדרישות החדשות

5. **"לא מקבל מייל איפוס"**
   - בדוק ב-Spam/Junk
   - ודא שהגדרות SMTP תקינות ב-Backend
   - הסתכל בלוגים של השרת לשגיאות
   - השתמש ב-`get_reset_token.py` בתיקיית Backend

## 🔐 אבטחה

### דרישות סיסמה
כל סיסמה חדשה חייבת לעמוד בדרישות הבאות:
- ✓ לפחות 8 תווים
- ✓ אות גדולה באנגלית (A-Z)
- ✓ אות קטנה באנגלית (a-z)
- ✓ ספרה (0-9)
- ✓ תו מיוחד (!@#$%^&* וכו')

**דוגמאות לסיסמאות תקינות:**
- `Aa123456!`
- `MyPass123@`
- `Secure#2025`

### תהליך איפוס סיסמה
1. לחץ "שכחתי סיסמא" במסך ההתחברות
2. הזן את כתובת המייל שלך
3. קבל מייל עם קוד איפוס (בדוק גם ב-Spam)
4. הזן את הקוד וסיסמה חדשה חזקה
5. התחבר עם הסיסמה החדשה

> 💡 **טיפ:** הקוד תקף ל-24 שעות בלבד!
